.. -*- mode: rst -*-

=================================
MScanner: Extend a set citations
=================================

:Author: Graham Poulter
                                      

The "doc" directory
-------------------

This directory contains documentation about MedScanner

* INSTALL.xhtml

  Documentation about installation, dependencies and how to
  obtain input data.

* epydoc.pyw, epydoc.prj

  Files for generating ePyDoc API documentation from the source
  (requires ePyDoc)
  
* html/

  HTML API for the program generated by ePyDoc.

The "lib" directory
-------------------

article.py -- Classes for managing articles and features
dbexport.py -- Export results to database
dbshelve.py -- Persistent dictionary with Berkeley DB
genedrug.py --  Extract gene-drug relations from abstracts
medline.py -- Update database of Articles and features
scoring.py -- Scores and reports for terms and documents
validation.py -- Cross-validation and performance reports
xmlparse.py -- Parse XML into article objects
templates/*.prep -- Preppy templates for reports

The "bin" directory
-------------------

Executable programs for the scanner

configuration.py -- Provide options and paths for update/query/validate
drugtable.py -- Parse drug table into a Pickle
meshexludes.py -- Parse MeSH descriptors for exclusions and synonyms
fixcorpora.py -- Make sure positives, negatives are unique, disjoint, non-bad
geneontology.py -- Extract PMIDs from GeneOntology annotations
query.py -- Query MedScanner for results 
update.py -- Add XML directory (or a Pickle) of Articles to database
validate.py -- Do cross-validation and output performance stats

The "test" directory
--------------------

Just some unit tests that are not part of the source files.
