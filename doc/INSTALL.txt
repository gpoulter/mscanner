=====================================================
MScanner: Retrieving Medline citations using examples
=====================================================

:Author: Graham Poulter
                                

MScanner takes a set Medline citations, and trains a
classifier to search for citations similar to the set as a whole.

Installation
============

Unzip the program, which extracts to the subdirectory ``mscanner``,
and install all the dependencies

Dependencies
============

`Python <http://www.python.org>`_
 Python programming language (version 2.5)

`Numpy <http://numpy.scipy.org>`_
 Fast vector operations.
 
`path <http://www.jorendorff.com/articles/python/path/>`_
 Path manipulation.

`Cheetah <http://www.cheetahtemplate.org>`_
 Templating engine for generating HTML output.

`Gnuplot-py <http://gnuplot-py.sourceforge.net/>`_
 Plots the cross validation output graphs. Gnuplot-py requires
 `Gnuplot <http://www.gnuplot.info/>`_ to be installed. 

`Epydoc <http://epydoc.sourceforge.net/>`_
 [optional] Generates API documentation for MScanner.

`SciPy <http://www.scipy.org>`_
 [optional] For Kernel Density Estimation (KDE) in place of histograms.

`web.py <http://webpy.org>`_
 [optional] Framework needed to run the MScanner web interface.


Input Data
==========

You will need to obtain the following data:

Medline
 A directory of xml.gz files for Medline citation records.
 Medline can be obtained by signing and faxing a memorandum of
 understanding with the NLM and downloading via FTP.

A list of PubMed IDs
 A text file with one PubMed ID per line.
